## API

### Props

| 参数 | 类型 | 默认值 | 说明 |
|-----|------|-------|-----|
| type | `TagTypes` | - | 类型 |
| round | `boolean` | - | 半圆 |
| href | `string` | - | 链接 |
| target | `_self`, `_blank` | - | 链接打开类型 |
| onClick | `() => void` | - | 点击事件 |
| onClose | `() => void` | - | 关闭事件 |
| className | `string` | - | 类名 |
| style | `React.CSSProperties` | - | 样式 |
