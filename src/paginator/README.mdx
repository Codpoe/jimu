## API

### Props

| 参数 | 类型 | 默认值 | 说明 |
|-----|------|-------|-----|
| total | `number` | - | 总数。必须 |
| pageSize | `number` | - | 页长。必须 |
| page | `number` | - | - | 当前页码 |
| defaultPage | `number` | `1` | 默认页码 |
| quickJump | `boolean` | - | 是否快速跳转 |
| onChange | `(page: number) => void` | - | 页码变化事件 |
| className | `string` | - | 类名 |
| style | `React.CSSProperties` | - | 样式 |
