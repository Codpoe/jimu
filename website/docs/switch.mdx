## 演示

### 简单示例

```jsx
export function Demo1() {
  const [checked, changeChecked] = React.useState(false);
  return (
    <>
      <Switch checked={checked} onChange={changeChecked} />
      <div>{String(checked)}</div>
    </>
  );
}
```


## API

### Props

| 参数 | 类型 | 默认值 | 说明 |
|-----|------|-------|-----|
| checked | `boolean` | - | 是否选中 |
| defaultChecked | `boolean` | `false` | 是否默认选中 |
| disabled | `boolean` | `false` | 禁用 |
| onChange | `(checked: boolean, target: SwitchProps) => void` | - | 变更事件 |
| className | `string` | - | 类名 |
| style | `React.CSSProperties` | - | 样式 |
