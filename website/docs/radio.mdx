## 演示

### 简单示例

```jsx
export function Demo1() {
  const [checked, changeChecked] = React.useState(false);
  return (
    <>
      <Radio
        checked={checked}
        onChange={changeChecked}
      >
        Apple
      </Radio>
      <div>{value}</div>
    </>
  );
}
```

## API

### Radio Props

| 参数 | 类型 | 默认值 | 说明 |
|-----|------|-------|-----|
| value | `any` | - | 值 |
| checked | `boolean` | - | 是否选中 |
| defaultChecked | `boolean` | `false` | 是否默认选中 |
| disabled | `boolean` | `false` | 禁用 |
| onChange | `(checked: boolean, target: RadioProps) => void` | - | 变更事件 |
| className | `string` | - | 类名 |
| style | `React.CSSProperties` | - | 样式 |

### Radio.Group Props

| 参数 | 类型 | 默认值 | 说明 |
|-----|------|-------|-----|
| value | `any` | - | 值 |
| defaultValue | `any` | - | 默认值 |
| disabled | `boolean` | `false` | 禁用 |
| onChange | `(value: any, target: RadioGroupProps) => void` | - | 变更事件 |
| className | `string` | - | 类名 |
| style | `React.CSSProperties` | - | 样式 |
