## 演示

### items

```jsx
export function Demo1() {
  const items = [
    { value: 'apple', label: 'Apple' },
    { value: 'smartisan', label: 'Smartisan' },
    { value: 'samsung', label: 'Samsung' },
  ];
  return (
    <Tab items={items} />
  );
}
```

<br />

indicator 指示器也可以接收一个函数`(value: any) => React.ReactNode`，自定义程度更高。

```jsx {7-11,17}
export function Demo5() {
  const items = [
    { value: 'apple', label: 'Apple' },
    { value: 'smartisan', label: 'Smartisan' },
    { value: 'samsung', label: 'Samsung' },
  ];
  const indicators = {
    apple: '👆',
    smartisan: '👉',
    samsung: '👈'
  };
  return (
    <Tab
      items={items}
      defaultActive="apple"
      indicator={
        value => <div style={{ marginBottom: '-10px' }}>{indicators[value]}</div>
      }
    />
  );
}
```

## API

### Tab Props

| 参数 | 类型 | 默认值 | 说明 |
|-----|------|-------|-----|
| items | `ItemProps[]` | - | 项 |
| active | `any` | - | 选中的项值 |
| defaultActive | `any` | - | 默认选中的项值 |
| indicator | `React.ReactNode`, `(value: any) => React.ReactNode` | - | 指示器 |
| stretch | `boolean` | - | 是否铺满 |
| onActiveChange | `(value: any, item: ItemProps) => void` | - | 变更事件 |
| className | `string` | - | 类名 |
| style | `React.CSSProperties` | - | 样式 |

### Tab.Item Props

| 参数 | 类型 | 默认值 | 说明 |
|-----|------|-------|-----|
| label | `React.ReactNode` | - | 标签 |
| value | `any` | - | 值 |
| indicator | `React.ReactNode` | - | 指示器 |
| active | `boolean` | - | 是否选中 |
| stretch | `boolean` | - | 是否铺满 |
| disabled | `boolean` | - | 是否禁用 |
| className | `string` | - | 类名 |
| style | `React.CSSProperties` | - | 样式 |
