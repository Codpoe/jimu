## API

### LoadingProps

| 参数 | 类型 | 默认值 | 说明 |
|-----|------|-------|-----|
| color | `string` | `currentColor` | 颜色 |
| colorType | `default`, `primary`, `success`, `warning`, `error` | `default` | 内置的颜色类型 |
| size | `string`, `number` | `1em` | 尺寸 |
| label | `React.ReactNode` | - | 标签 |
| vertical | `boolean` | `false` | 是否垂直排列 |
| className | `string` | - | 类名 |
| style | `React.CSSProperties` | - | 样式 |

### Loading.Wrap Props

| 参数 | 类型 | 默认值 | 说明 |
|-----|------|-------|-----|
| loading | `boolean` | `false` | 是否加载中 |
| mask | `boolean` | `true` | 是否显示蒙层 |
| className | `string` | - | 类名 |
| style | `React.CSSProperties` | - | 样式 |
