# Redefy

[![npm](https://img.shields.io/npm/v/redefy)](https://www.npmjs.com/package/redefy)

简单、易用的平民级 React 组件库

## ✨ 特性

- 🔥 使用 TypeScript 开发
- 📦 支持按需引入
- 🌝API 设计简单易用（自我感觉良好）

## 🤔 初稿

这个组件库其实是很早之前由于毕设项目的需要而开发的，功能比较简单，但还是能大致满足当时毕设的小后台项目需求。
在此之后这个组件库代码基本荒废，虽然一直想着继续完善，但心有余而力不足，加班到心累...

去年决定开始用 TS 来重构，并且重写了组件编译打包的逻辑，起码整体看起来更像一个**作品**。
